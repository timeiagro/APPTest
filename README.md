## Como Usar

### Default

#### Yarn

```bash
  yarn install
```
#### Npm

```bash
  npm i
```


### Ios

ir para para a pasta /ios e executar:

```bash
  pod install
```

Em seguida voltar para pasta /root e executar:

```bash
  react-native run-ios --simulator="iPhone 11" (Simulador de sua preferência)
```

### Android

Executar:

```bash
  react-native run-android
```
